JsDoc is a program for automating API creation. It works through a server-client system. After you have installed JsDoc, you only have to worry about your code and its documentation. When you write your source file, you have its API immediately available.

  • Load the page skeleton.
  • Ask Server class for session id.
    • If session id exists and is not expired ...
    • Otherwise, application asks Server class if application was intialized.
      • If application was intialized, it asks user for password.
        • If password is correct, it sets the session id and restarts the page.
        • Otherwise, application reloads itself and asks for password again. After three fails a captcha is activated.
      • If application was not initialized, it asks user for generating the admin password.
        • If application is installed, it sets the admin password and restarts the page.
        • Otherwise, application fails sending to console the error message UNREGISTERED.
